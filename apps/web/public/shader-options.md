# background shader options

candidates for the landing page ambient background. all available in the `shaders` npm package.

1. **SimplexNoise** — slow-moving organic noise, very subtle
2. **FloatingParticles** — drifting particles, space-like (current choice)
3. **Strands** — flowing fiber/thread effect
4. **Aurora** — gentle northern lights glow
5. **FlowField** — smooth directional movement
6. **Plasma** — soft color blending
